# COVID LFT Cassette Detector (YOLOv8)

## Install

**Python 3.8+** — https://www.python.org/downloads/ (check "Add Python to PATH")

Then run:
```
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
pip install ultralytics
```

---

## Before running — change these paths

Open these 2 files and replace `c:/Users/yahia/OneDrive/Desktop/school/DeepLearning` with your own path:

| File | Line | What to change |
|---|---|---|
| `setup_dataset.py` | 3 | `BASE = "your/path/here"` |
| `train_model.py` | 7 | `data="your/path/here/dataset.yaml"` |

`dataset.yaml` is auto-generated by `setup_dataset.py`, so no need to edit it manually.

---

## Run in order

```
python convert_to_yolo.py
```
Converts COCO annotations to YOLO `.txt` label files.

```
python setup_dataset.py
```
Splits images into train/val and creates `dataset.yaml`.

```
python train_model.py
```
Trains the model. Results saved to `runs/covid_lft/`.

```
python evaluate_model.py
```
Runs predictions on the `test/` folder. Output saved to `runs/evaluate/test_predictions/`.

> Before running evaluate, update the model path in `evaluate_model.py` to your actual `best.pt` (found in `runs/detect/.../weights/best.pt`).
